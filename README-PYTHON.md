# QR Code Generator - Python Flask Version

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Run

### Windows:
```bash
run.bat
```

### Linux/Mac:
```bash
python app.py
```

## Access

Open browser: http://localhost:5000

## Features
- Custom colors
- Multiple sizes
- Different styles
- Download QR codes
- Dark/Light theme
